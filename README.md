## How to use
`JSHell(levelx,input,wrapWithEval,runInParentScope,debug_mode)`
+ LevelX : 0 | 50 | 100
> Strong Level of JSHell
+ wrapWithEval : true | false
+ runInParentScope : true | false
+ debugMode (Unsecure): true | false

`E.x` : `JSHell('50',"alert(1)",true,true,false)`
